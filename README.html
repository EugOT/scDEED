dubiouspack

This package provides functions to detect dubious cells and trustworthy cells under tSNE and UMAP. Furthermore, based on the number of dubious cells, functions in this package will provide the best perplexity parameter under tSNE and best n.neighbors parameter under UMAP.

Installation

You can install the released version of dubiousdetector from CRAN with:

install.packages("dubiousdetector")

Example

This is a basic example which shows you how to find the best parameter. We use pmbc data as a demo:

# install.packages("dubiousdetector")
suppressPackageStartupMessages(library(dubiousdetector))
data(pbmc.data)

Choose the suitable dimension (num_pc)

chooseK(pbmc.data)

Example for umap

umap_example <- umap_tsne_process(pbmc.data , num_pc = 10, use_method = "umap",visualization = TRUE)

Example for tsne

tsne_example <- umap_tsne_process(pbmc.data, num_pc = 10, use_method = "tsne",visualization = TRUE)
